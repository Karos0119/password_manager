## Building

Install rustup on your system https://rustup.rs/

Clone the repo

```git clone https://github.com/Karos0119/password_manager```

Go into the directory and build

```cd password_manager```

```cargo build -r```


The executable will be in ```target/release```

## Usage

```> password_manager```
